# Auto-Browser
Made an Auto-Browser that browses automatically to what is predefined in the code. It is helpful for testing industry-grade projects.
## What the auto-browser does-
* It first connects to the browser you want through a Python library named selenium.
* It then browses through the steps written inside the code.
## What is it currently doing?
### Showing me my timetable-
It is browsing Google and searching for my college name, and according to the search results finding the college name in search results and then clicks on it, and find the department section clicks on Information Technology, and then click on Time Tables, then the required section time table, and shows me the correct section time table to which I belong.
## Future Scope-
Making this auto browser feasible to any project so that it can become capable of testing any project with data.
